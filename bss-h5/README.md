### 默认文件
| 文件名/格式                    | 描述                                                 |
|---------------------------|----------------------------------------------------|
| default.ts                | 默认插槽                                               |
| [slug]                    | slug 是动态参数, 动态参数段                                  |
| [...slug]                 | slug 是动态参数, 是一个数组                                  |
| /shop/[[...slug]]         | slug 是动态参数, 会收集包含/shop的所有子路径                       |
| error.tsx                 | 错误页面                                               |
| global-error.tsx          | 全局错误页面                                             |
| instrumentation.js        | 集成可视化工具                                            |
| instrumentation-client.js | 集成前端可视化工具                                          |
| intercepting-routes.js    | 路由拦截器: 允许在当前布局中加载其他部分的路由 (.)当前级别 (..)上一级别 (...)根目录 |
| layout.js                 | 当前以及子目录的布局                                         |
| loading.js                | 当前以及子目录的记载组件                                       |
| middleware.js             | 中间件 客户端到服务器端的中间件                                   |
| not-found.js              | 404                                                |
| global-not-found.js       | 404                                                |
| @team                     | 平行路由, 其实就是有名字的插槽内容                                 |
| public                    | 静态文件, 以/开始饮用文件                                     |
| route.js                  | 路由文件, 编码方式路由                                       |
| (folderName)              | 路由组                                                |
| template.js               | 类似layout文件, 但是可以用 userEffect,重置子客户端状态,更改默认框架行为     |
